# DesertBloom Labs 🌵✨

**Innovación, Software y Transformación Digital desde San Luis Potosí, México.**

---

### 👋 ¡Hola! Somos DesertBloom Labs

Somos una startup de desarrollo de software enfocada en crear soluciones SaaS de alto impacto, automatización inteligente y experiencias digitales modernas. Combinamos diseño de producto excepcional con arquitectura técnica robusta para ayudar a negocios y emprendedores a florecer en el ecosistema digital.

### 🚀 Proyectos Destacados

*   **[Tu Cascarita](https://github.com/DesertBloom-Labs/tu-cascarita-frontend)**: Plataforma integral para la gestión de ligas y torneos de fútbol amateur. (Next.js, TypeScript, Tailwind CSS, Three.js).
*   **[PlayNetix](https://github.com/DesertBloom-Labs/PlayNetix)**: Hub y tienda para desarrolladores de videojuegos indie. (Angular, Node.js).
*   **[MN Consulting](https://github.com/DesertBloom-Labs/mn-consulting-frontend)**: Dashboard de analítica y newsletter integrado con React Email. (React, Vite, Tailwind v4).
*   **SkAI**: Aplicación de análisis climático y ambiental (Finalista NASA Space Apps Hackathon).
*   **[The Agency](https://github.com/DesertBloom-Labs/agency-agents)**: Colección especializada de perfiles de agentes de IA para flujos de trabajo avanzados.
